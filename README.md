# Schedule

This is a Schedule desktop app. It shows the schedule for 1 month, as each employee added has a information on each date of what shift and duties it is.

## :hammer_and_pick: Used Technologies and languages
- C#
- WPF
- Entity Framework
- MS SQL Server
- XAML
